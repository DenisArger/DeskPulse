# Project Memory

## Project Name
DeskPulse

## Назначение проекта
Проект содержит персональные Python-скрипты для desktop-автоматизации в Linux-среде (X11 + PipeWire/PulseAudio).

## Ключевые цели
1. Быстрое управление окнами рабочего стола.
2. Автоматическое восстановление звука при нестабильной маршрутизации аудио.
3. Управление громкостью колесиком мыши в зоне панели.
4. Надежное подключение Bluetooth-наушников без ручного дебага.

## Технический контекст
- ОС: Linux desktop
- Графика: X11
- Аудиостек: PipeWire с совместимостью PulseAudio (`pactl`), низкоуровнево `amixer`
- Bluetooth: BlueZ (`bluetoothctl`)
- Скрипты: Python 3

## Известные ограничения
- Без расширения X RECORD wheel-скрипт работает только через grab fallback (`--allow-grab`), что может мешать прокрутке в приложениях.
- Поведение аудиопортов зависит от физического разъема (headphones/lineout) и определения доступности портов драйвером.

## Рабочие практики
- Для постоянной стабилизации звука использовать `headphones_guard.py` в режиме watch.
- Скрипт колесика включать по необходимости (через toggle/горячую клавишу), а не держать постоянно, если это мешает приложениям.
